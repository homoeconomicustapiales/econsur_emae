# 📊 EMAE Argentina Dashboard

Dashboard interactivo del EMAE (INDEC) construido con:

- Node.js
- Lightweight Charts
- Excel → JSON pipeline

## 🚀 Features

- Visualización profesional tipo trading
- Zoom y navegación temporal
- Exportación a PNG
- Backend API simple

## ⚙️ Instalación

### Backend
```bash
cd backend
npm install
node convert.js
node server.js
